# Арифметические выражения

Вычисление арифметических выражений с заведением переменных.
Используйте целочисленные переменные.

### Пример:

##### a = 1 - 2 - 3;
##### b = 2 + 3;
##### a = b * 3;
##### a = -a * (a / 2);
##### c = a + 1;
##### d = 2^3 ^2;
##### e = d + 1;
##### r = 2^3^3;

### Вывод:

##### a = -4;
##### b = 5;
##### a = 15;
##### a = -105;
##### c = -104;
##### d = 512;
##### e = 513;
##### r = 134217728;
